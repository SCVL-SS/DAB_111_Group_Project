Data Source and Purpose

This website provides a comprehensive overview of League of Legends champions, featuring their base statistics and roles.

Data Breakdown

The dataset includes the following information for each champion:

  • Champion Name: The name of the champion
  • Role: The primary role in the game (Top, Mid, ADC, Support, Jungle)
  • Base Health: Starting health points
  • Base Mana: Starting mana points
  • Base Armor: Starting armor value
  • Base Attack Damage: Initial attack damage
  • Gold Efficiency: Creep Score per second

Data Disclaimer

This data represents base statistics and does not reflect in-game upgrades, items, or specific champion abilities. The information is intended for reference and analysis purposes.
